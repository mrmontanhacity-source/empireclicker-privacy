Política de Privacidade
Empire Clicker Pro
Última atualização: 16 de fevereiro de 2026

1. Introdução
A AppRs ("nós", "nosso" ou "nos") opera o aplicativo móvel Empire Clicker Pro (o "Aplicativo"). Esta página informa sobre nossas políticas em relação à coleta, uso e divulgação de dados pessoais quando você usa nosso Aplicativo.

Usamos seus dados para fornecer e melhorar o Aplicativo. Ao usar o Aplicativo, você concorda com a coleta e uso de informações de acordo com esta política.

2. Informações que Coletamos
2.1 Dados de Uso
Podemos coletar informações sobre como o Aplicativo é acessado e usado ("Dados de Uso"). Esses Dados de Uso podem incluir informações como:

Tipo e modelo do dispositivo
Versão do sistema operacional
Identificador único do dispositivo
Endereço IP
Tempo gasto no aplicativo
Páginas visitadas dentro do aplicativo
Interações com recursos do jogo
2.2 Dados do Jogo
O Aplicativo armazena localmente no seu dispositivo:

Progresso do jogo (dinheiro, geradores, upgrades)
Conquistas desbloqueadas
Estatísticas de jogo
Configurações e preferências do usuário
Importante: Estes dados são armazenados apenas localmente no seu dispositivo e não são enviados para nossos servidores.

3. Publicidade
3.1 Google AdMob
Utilizamos o Google AdMob para exibir anúncios no Aplicativo. O AdMob pode coletar e processar informações sobre você, incluindo:

Identificador de publicidade do dispositivo (IDFA/GAID)
Endereço IP
Dados de localização aproximados
Interações com anúncios
Para mais informações sobre como o Google usa dados, consulte a Política de Privacidade do Google.

3.2 Anúncios Personalizados
O Google AdMob pode usar cookies e tecnologias similares para fornecer anúncios personalizados com base nos seus interesses. Você pode optar por não receber anúncios personalizados através das configurações do seu dispositivo:

Android: Configurações → Google → Anúncios → Desativar personalização de anúncios
iOS: Configurações → Privacidade → Publicidade → Limitar rastreamento de anúncios
4. Como Usamos os Dados
Utilizamos os dados coletados para:

Fornecer e manter o Aplicativo
Melhorar a experiência do usuário
Exibir anúncios relevantes
Analisar o uso do Aplicativo
Detectar, prevenir e resolver problemas técnicos
Cumprir obrigações legais
5. Compartilhamento de Dados
Não vendemos, trocamos ou transferimos suas informações pessoais para terceiros, exceto:

Google AdMob: Para exibição de anúncios
Requisitos legais: Quando exigido por lei ou para proteger nossos direitos
6. Segurança dos Dados
A segurança dos seus dados é importante para nós. Implementamos medidas de segurança comercialmente aceitáveis para proteger suas informações. No entanto, nenhum método de transmissão pela Internet ou armazenamento eletrônico é 100% seguro.

7. Privacidade de Crianças
Nosso Aplicativo não coleta intencionalmente informações pessoalmente identificáveis de crianças menores de 13 anos. Se você é pai/mãe ou responsável e está ciente de que seu filho nos forneceu dados pessoais, entre em contato conosco.

8. Links para Sites de Terceiros
O Aplicativo pode conter links para sites de terceiros. Não temos controle sobre e não assumimos responsabilidade pelo conteúdo, políticas de privacidade ou práticas de sites ou serviços de terceiros.

9. Alterações nesta Política
Podemos atualizar nossa Política de Privacidade periodicamente. Notificaremos você sobre quaisquer alterações publicando a nova Política de Privacidade nesta página e atualizando a data de "Última atualização".

Recomendamos que você revise esta Política de Privacidade periodicamente para quaisquer alterações.

10. Seus Direitos
Você tem o direito de:

Acessar os dados que coletamos sobre você
Solicitar correção de dados imprecisos
Solicitar exclusão de seus dados
Optar por não receber anúncios personalizados
Retirar seu consentimento a qualquer momento
11. Retenção de Dados
Reteremos seus dados apenas pelo tempo necessário para os fins estabelecidos nesta Política de Privacidade. Os dados do jogo armazenados localmente permanecerão no seu dispositivo até que você desinstale o Aplicativo ou limpe os dados manualmente.

12. Conformidade com LGPD e GDPR
Este Aplicativo está em conformidade com:

LGPD (Lei Geral de Proteção de Dados - Brasil)
GDPR (General Data Protection Regulation - União Europeia)
